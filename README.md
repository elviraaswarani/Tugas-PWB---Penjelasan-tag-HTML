# Tugas-PWB---Penjelasan-tag-HTML
1.  DOCTYPE html : Identitas dokumen agar browser tahu ini adalah standar HTML5 terbaru.
2.  html : Akar utama yang membungkus semua kode dari awal sampai akhir.
3.  head : Area informasi teknis atau data "belakang layar" sebuah halaman web.
4.  title : Nama yang muncul di tab browser saat kamu membuka website tersebut.
5.  body : Tempat semua konten utama (teks, gambar, video) yang bisa dilihat pengunjung.
6.  h1 & h2 : Heading atau judul; h1 untuk judul paling besar, h2 untuk sub-judul di bawahnya.
7.  p : Paragraf untuk menulis teks atau narasi biasa.
8.  a : Link atau tautan untuk menghubungkan halaman satu ke halaman lainnya.
9.  img : Elemen untuk memasukkan dan menampilkan gambar pada halaman.
10. ul & li : Daftar poin; ul sebagai pembungkus list, dan li sebagai butir-butir isinya.
